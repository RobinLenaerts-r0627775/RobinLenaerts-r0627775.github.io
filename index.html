DiscordBot

https://github.com/MarioAquino-r0635688/discordbot
In het eerste jaar van mijn opleiding aan de UCLL heb ik samen met een medestudent een bot gemaakt voor de populaire communicatie app Discord. We hebben de bot in Java geschreven, wat achteraf gezien niet ideaal was maar het was vooral een leerervaring.

IoT project

https://github.com/MarioAquino-r0635688/Projectweek2
Tijdens de laatste projectweek van de UCLL-opleiding hebben we (een groep van 4 2de jaars studenten) op een week tijd met een raspberry pi en een webcam een people counter gemaakt. We hebben dit project ontwikkeld op basis van een open source python script. Dit was echter verre van perfect voor ons project dus hebben we redelijk code bijgescreven.

Technology in Healthcare Education: DementiApp

https://github.com/RobinLenaerts-r0627775/DementiApp
Als afstudeerproject heb ik met 2 medestudenten een applicatie ontwikkeld voor tablets. De applicatie is bedoeld om de levenskwaliteit van dementiepatiënten te verhogen door sociaal contact te bevorderen met behulp van reminiscentie. De applicatie is ontwikkeld met Xamarin.

VR experience voor AB-Inbev on-boarding (stage)


Als stage heb ik 3 maanden bij AB-Inbev gewerkt aan een Virtual Reality applicatie die gebruikt wordt bij on-boarding om het bedrijf beter te leren kennen. De applicatie is gemaakt met Unity en C# scripts. AB-Inbev was zeer tevreden van het eindresultaat.
